# build8-admin

Admin panel for Oud Restaurant.  
Built with React and Vite for fast development and hot module replacement (HMR).

## Features

- User management
- Order tracking
- Product updates
- Admin dashboard

## Tech Stack

- [React](https://reactjs.org/)
- [Vite](https://vitejs.dev/)
- [ESLint](https://eslint.org/)

## Getting Started

```bash
npm install
npm run dev
